# Rust Donation API

API simples para gerenciamento de doações em Rust.

## Objetivos
- Trabalhar com estado compartilhado em memória
- Criar rotas GET e POST
- Receber e armazenar doações temporariamente
- Praticar arquitetura básica de APIs

## Funcionalidades atuais
- POST /donate
- GET /total
- GET /donations

## Stack utilizada
- Rust
- Axum
- Tokio
- Serde

## Executando localmente

```bash
cargo run
